# GO-DO-LIST

Um projeto de estudos de uma To-Do-List com uma arquitetura profissional usando Golang + PostgreSQL + Gorilla WebSocket + Next.js.<br/>
Este repositório tem foco em arquitetura limpa, boas práticas, drag-and-drop, ordenação de dados, e foi desenvolvido sem fins lucrativos.<br/>
Este projeto utiliza Go no backend, PostgreSQL como banco de dados e Next no frontend, explorando conceitos modernos como drag-and-drop, comunicação em tempo real e organização modular.

## Objetivos

- Autenticação com JWT;
- Estrutura baseada em Clean Architecture;
- Simulação de features de produtos reais (ex: Trello, Jira);
- Criação de boards, colunas dinâmicas, cards com drag-and-drop, labels e etc;
- Escalabilidade horizontal e manutenção facilitada;
- Desenvolver um ambiente com PostgreSQL;
- Colaboração em tempo real (WebSocket).
